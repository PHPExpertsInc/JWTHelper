# JWTHelper

JWTHelper is an accessory library for Tymon's JWTAuth library.

## Usage

    // Attempts to authenticate a user's bearer token.
    JWTHelper::authenticate();

## Installation

Via Composer

``` bash
$ composer require phpexperts/jwt-helper
```

## Testing

``` bash
$ phpunit
```
